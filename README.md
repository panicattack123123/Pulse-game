## Как запустить проект локально
1. Скачайте и установите актуальную версию [Ren'Py SDK](https://www.renpy.org/).
2. Склонируйте этот репозиторий в папку проектов Ren'Py:
```bash
   git clone https://github.com/panicattack123123/Pulse-game.git
```
3. Откройте Ren'Py SDK, выберите проект Pulse-game в списке и нажмите «Запустить проект».
